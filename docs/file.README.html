使い方

Ruby 2.6 以上 でテストしています。

$ gem install pebbles-totsugen
これで `totsugen.rb` というのが $PATH に入ります。やばいですね。

ターミナルから

引数で渡したり

$ totsugen.rb 突然の死
＿人人人人人人＿
＞　突然の死　＜
￣Y^Y^Y^Y^Y￣

標準入力で渡したり

$ echo '突然のSTDIN' | totsugen.rb
＿人人人人人人人＿
＞　突然のSTDIN　＜
￣Y^Y^Y^Y^Y^Y￣

ライブラリとして（なんに使うんだこんなもん）

require 'pebbles/totsugen'
